# Joule interview

This repository aims to emulate a blog post application. It is imperfect on purpose, we'd like your comments on it !

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Prisma

# migrate

```bash
$ npx prisma migrate dev --name "<NAME>"
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
